# Excel Sales Analytics Dashboard

## Project Overview
This project analyzes retail sales data using Microsoft Excel to generate business insights through interactive dashboards.

The goal of this project is to demonstrate data analytics skills including data cleaning, pivot table analysis, KPI calculations, and dashboard creation.

## Dataset Information

The dataset contains **9,994 sales records** with **21 columns** representing orders, customers, products, and financial metrics.

### Key Columns
- Order ID
- Order Date
- Ship Date
- Ship Mode
- Customer ID
- Customer Name
- Segment
- Country
- City
- State
- Region
- Product ID
- Category
- Sub-Category
- Product Name
- Sales
- Quantity
- Discount
- Profit

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- KPI Calculations
- Interactive Dashboard Design

## Dashboard Features

The Excel dashboard provides the following analytics:

### Key Performance Indicators (KPIs)

- Total Sales
- Total Profit
- Margin %
- Total Quantity 


### Visualizations

- Sales by City
- Sales by Region
- Sales by Segment %
- Top 10 Product Sales
- shipmode %
- Qty by Category %
- Sales Category %
- Monthly Sales Trend
- Daily Sales
